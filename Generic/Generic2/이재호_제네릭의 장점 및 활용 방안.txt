제네릭은 메서드와 클래스에서 각각 데이터 형식이 다를 경우 사용하며
데이터 형식을 일반화 하여 다양한 형식을
처리하도록 도와준다 그로 인해 컴파일 할 때, 안정성을 보장해준다.

제네릭은 where 과 new 키워드를 사용하여 형식을 제약할 수 있으며, 
where 키워드를 사용할 경우 형식 매개 변수의 기능을 지정하여 제약 형식의 작업
및 메서드 호출을 사용할 수 있게 해준다.

new 키워드를 사용할 경우 new T() 의 컴파일 에러를 방지해준다. T 객체가
T() 생성자를 가지고 있다고 보장해주는 역할 이기 때문이다.

ArrayList의 경우 내부에 object 배열을 가지고 있기때문에 값 자체를 저장하는 것이 아닌 
object 타입으로 박싱된 데이터의 주소를 배열에 저장하는 방식이다.
List<T>를 사용할 경우 내부에 T 타입의 배열을 따로 가지고 있기때문에 
값 형식의 리스트를 선언할 경우 해당 값을 그대로 복사하여 박싱이 일어나지 않게 된다.
